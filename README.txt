FR
Dans le cadre d'un projet universitaire, ce jeu est développé pour valider des connaissances vu en java.

Les commandes principales que vous pouvez utiliser :

- "sh scripts/run.sh pour lancer le programme.

- "sh scripts/makejar.sh" pour créer un jar contenant tous les .class

- "sh scripts/makedoc.sh" pour générer la javadoc (le fichier principal est alors doc/index.html)

- "sh scripts/clean.sh" pour supprimer tous les fichiers générés.(doc jar et build)



EN
This program is developped during university's project in order to validate studied knowledges in java.

The commands you can use :

- "sh scripts/run.sh to launch the program.

- "sh scripts/makejar.sh" create a .jar file containing all .class program file.

- "sh scripts/makedoc.sh" generate the javadoc (the main file is doc/index.html)

- "sh scripts/clean.sh" to delet all generated files générés.(doc jar and build)
